# README
    该文档用来说明晓道消息项目爬虫部分更新情况
## 项目总体架构
1. **爬虫部分**
---
>该部分为项目的主体部分，主要为客户所需要的各种爬虫主文件

2. **反反爬策略**
---
>该部分应对有反爬虫策略的网站，主要包括但不限于：
- user-agent池
- proxy池
- 验证码识别
- cookie池

3. **DB**
---
>该部分用于爬取后的数据入库


4. **logger**
>该部分用于存储日志
